rule30.visualize¶

rule30.visualize.image_from_matrix(matrix, block_size=1)[source]¶

Generates an image from a 2d boolean matrix.

The matrix can be any data structure that supports len on both dimensions and can be indexed by two subscripts (which is then treated as a bool).

Each element is drawn as a square of the specified block_size, where 0 is drawn in white and 1 is drawn in black.

Parameters:

matrix : Sequence[Sequence[Any]]

block_size : int, optional
Returns:

PIL.Image.Image
